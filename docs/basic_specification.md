## 1. 画面構成

アプリはボトムナビゲーションにより以下の主要タブへ遷移する。

| タブ名 | 画面名 | 概要 |
|--------|--------|------|
| 計測タブ | 計測準備画面 | 車両選択・環境設定→計測開始 |
| 計測タブ | 計測中画面 | 計測実行・停止 |
| 計測タブ | 計測結果詳細 | グラフ・最高馬力・メモ表示 |
| リアルタイム | リアルタイム画面 | 馬力・速度・GPS情報のライブ表示 |
| 記録タブ | 記録一覧 | 全計測ログの一覧表示 |
| ガレージ | ガレージ画面 | 登録車両一覧 |
| ガレージ | 車両設定画面 | 車両情報の登録・編集 |

---

## 2. 各画面仕様

### 2-1. 計測準備画面

計測タブへ切り替えた際に最初に表示される画面。

- 画面上部に画面名を表示（アプリ名は非表示）
- 計測に使用する車両をプルダウン等で選択
- 気温（℃）を任意入力（デフォルト値なし）
- 大気圧（hPa）を任意入力（デフォルト値なし）
- 画面下部に「計測開始」ボタンを配置し、押下すると計測中画面へ遷移

---

### 2-2. 計測中画面

計測実行中に表示される画面。

- 画面上部に画面名を表示（アプリ名は非表示）
- 詳細ロギングカードは非表示
- 「計測停止」ボタンを押下すると計測を終了し、計測結果詳細画面へ自動遷移
- 【Pro Mode】トルクのリアルタイム表示

---

### 2-3. 計測結果詳細

計測終了後に表示される結果画面。

- 画面上部に画面名を表示（アプリ名は非表示）
- 計測日時を小さく表示
- 気温・大気圧（入力された場合のみ表示、デフォルト値なし）
- タイヤサイズを入力可能
- 駆動ロス係数を入力・修正可能
- グラフ：縦軸＝馬力、横軸＝時間経過
- 最高馬力を表示
- メモ欄（自由記述・編集可能）
- 計測時の車両情報をカードで表示（初期：車名・重量のみ、タップで詳細展開）
- 車両カード内にもメモ欄を設け編集可能
- 【Pro Mode】グラフ横軸を回転数へ切り替えるトグルを表示
- 【Pro Mode】トルクグラフを表示

---

### 2-4. リアルタイム画面（ダッシュボード代替）

走行中の各種情報をリアルタイムで確認できる常時表示画面。

- 馬力をリアルタイム表示
- 速度をリアルタイム表示
- GPS情報（緯度・経度・標高）を表示
- 速度のリアルタイム表示は本画面のみで行う（計測中画面では非表示）

---

### 2-5. 記録一覧

過去のすべての計測記録を一覧で表示する画面。

- 各記録に計測日時を表示
- 各記録に最高馬力を表示
- タップすると計測結果詳細画面へ遷移

---

### 2-6. ガレージ画面

登録されている車両の一覧を表示する画面。

- 登録済み車両をカード形式で一覧表示
- カードには車名・重量をデフォルト表示
- カードをタップすると車両設定画面へ遷移
- 【Pro Mode】複数車両の登録・管理が可能

---

### 2-7. 車両設定画面

車両情報を登録・編集する画面。

- 画面上部に画面名を表示（アプリ名は非表示）
- 入力項目：車名、車両重量
- メモ欄（自由記述・編集可能）
- 【Pro Mode】複数車両の登録が可能
- 【Pro Mode】ギア比の登録が可能（1速〜8速 + ファイナルギア比）

---

## 3. 計測機能仕様

### 3-1. 基本フロー

1. 計測タブへ切り替え → 計測準備画面へ自動遷移
2. 車両を選択し、気温・大気圧を任意入力
3. 「計測開始」ボタンを押下 → 計測中画面へ遷移
4. 車を加速させながら計測を実施
5. 「計測停止」ボタンを押下 → 計測結果詳細画面へ自動遷移
6. 結果確認・メモ入力・グラフ確認

---

### 3-2. 計測データ項目

| データ項目 | 入力方式 | 備考 |
|-----------|---------|------|
| 計測日時 | 自動記録 | 計測開始時のタイムスタンプ |
| 馬力 | センサー計測 | 時系列グラフとして保存 |
| 最高馬力 | 自動算出 | 計測中の最大値 |
| 気温（℃） | 手動入力（任意） | デフォルト値なし |
| 大気圧（hPa） | 手動入力（任意） | デフォルト値なし |
| タイヤサイズ | 手動入力（任意） | |
| 駆動ロス係数 | 手動入力・修正 | 補正計算に使用 |
| メモ | 手動入力（任意） | 自由記述、後から編集可 |
| 関連車両情報 | 選択 | 計測時に選択した車両データを紐付け |
| トルク | センサー計測 | 【Pro Mode】 |

---

## 4. 車両管理機能仕様

### 4-1. 登録可能な車両情報

| 項目 | 必須/任意 | 備考 |
|------|---------|------|
| 車名 | 必須 | |
| 車両重量 | 必須 | |
| メモ | 任意 | 自由記述・後から編集可 |
| ギア比（1〜8速） | 任意 | 【Pro Mode】 |
| ファイナルギア比 | 任意 | 【Pro Mode】 |

### 4-2. 車両カード表示仕様

- 初期表示：車名・車両重量のみ表示
- タップで詳細情報を展開表示
- 詳細情報にはメモ欄を含む（編集可能）

---

## 5. データ管理仕様

- データベース：SQLite（端末ローカル保存）
- 計測データ・車両データはすべてローカルDBに保存
- 外部サーバへのデータ送信は行わない
- 計測記録には関連車両情報を紐付けて保存

---

## 6. Pro Mode 機能一覧

以下の機能は課金プラン（Pro Mode）でのみ利用可能とする。

| 機能 | 内容 |
|------|------|
| リアルタイムロギング | 計測中にトルクをリアルタイム表示・ログ保存（馬力は無料版でも表示） |
| トルク表示 | 計測結果詳細でトルクグラフを表示 |
| グラフ軸切替 | 計測結果詳細のグラフ横軸を回転数へ切り替え可能 |
| 複数車両管理 | 2台以上の車両を登録・管理 |
| ギア比登録 | 1〜8速＋ファイナルギア比を車両設定に登録 |
